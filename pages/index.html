简介

Tabris.js是一个移动开发框架，可以使用完全用 JavaScript 或 TypeScript + JSX 实现的一套代码，来开发iOS、Android 和 Windows app。 Tabris.js提供了访问原生控件和原生方法的接口，并且在每个平台上提供原生应用的性能和外观。

Tabris.js以兼容性和可扩展性为中心来开发。它暴露了各种常见的JavaScript API，比如 Timers, Console, Fetch 和 WebSockets。 你可以在Tabris.js中使用npm中已有的库，也可以使用Cordova插件来访问Tabris.js没有直接支持的设备方法。

第一步

你不需要在你的电脑中安装任何东西就可以使用Tabris.js。

  • 在你的设备上安装Tabris.js开发者App，然后浏览其中的例子。
  • 在Tabris.js 平台在线编辑Tabris.js app的JavaScript代码。
  • 用你的开发者app扫描平台页面中的二维码，加载你编辑的app。

参考Quick Start Guide来开始开发你的Tabris.js app。 我们也有一个优秀的 ebook 来说明如何创建、部署和测试你的第一个Tabris.js app。
