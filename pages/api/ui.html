ui

继承自 Composite

是所有控件的父级。可通过其访问状态栏、导航栏、抽屉控件和内容视图。

通过 “const {ui} = require('tabris');” 引入该对象。

属性

contentView

Type: Composite

内容视图是构成应用主UI的控件的容器。覆盖整个应用界面。可将控件直接添加到内容视图中。
此属性只能在创建控件时设置。一旦设置，不能修改。

drawer

Type: Drawer

表示可以从屏幕左侧划入的抽屉控件。默认被锁定不可使用。想要使用的话，设置locked属性为false。此控件内可包含任意类型的控件。
此属性只能在创建控件时设置。一旦设置，不能修改。

navigationBar

Type: NavigationBar

表示屏幕上显示后退、主页或其他相似按钮的系统导航栏。仅在安卓系统可用。
此属性只能在创建控件时设置。一旦设置，不能修改。

statusBar

Type: StatusBar

表示系统状态栏。是显示通知、状态图标和设备时间的区域。
此属性只能在创建控件时设置。一旦设置，不能修改。
